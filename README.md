# Social Network Website MiniProject - Socialio

**มินิโปรเจกต์ เว็บไซต์โซเชียลมีเดีย โซเชียลลิโอ้ สร้างโดยใช้**
- React
- MERN Stack

**โดยภายในเว็บไซต์จะประกอบไปด้วยระบบพื้นฐานต่าง ๆ และมีระบบตามที่ถูกให้ Requirement ไว้**
- หน้าสมัครสมาชิก
- หน้าล็อคอินเข้าสู่ระบบ
- หน้าลืมรหัสผ่าน
- หน้า Timeline เรียงตามวันและเวลา
- สามารถโพสต์รูป / ข้อความ / วิดีโอได้
- ระบบ Like / Comment
- ระบบเพิ่มเพื่อน
- ระบบแชท
- สามารถสร้างกลุ่มได้
